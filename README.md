## Technologies

Project is created with:

* HTML5: 5.3
* SASS: 1.49.8
* Jquery: 3.6.0

## Setup

To run this project, install it locally using [NodeJS](https://nodejs.org/en/download/):

```
$ cd ../web-starter-stable
$ npm install
$ npm run website
```
